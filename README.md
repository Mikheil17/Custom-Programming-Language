# Custom Programming Language
The core of any Computer Science degree

## What It Is
A TIPS interpreter, built from scratch

## Tech
C++

## Key Features
Variable declaration, assignment, and memory management
Arithmetic and boolean expression evaluation
Logical and relational operators (AND, OR, NOT, <, >, ==, etc.)
Control flow: IF/THEN/ELSE, WHILE loops
Compound assignment operators
Custom Spawn statement
Descriptive error handling for invalid syntax

## Context
Created as a capstone project for Theory & Implementation of Programming Languages class

## How to Run
 
1. Download all files into the same folder on your computer.
2. Open a WSL terminal in that folder.
3. Build the interpreter (run once): `make`
4. Run it against a test file: `./parse <path-to-test-file>`
5. Sample test programs are included in `tests_part1/` through `tests_part4/`, covering lexing, parsing, and evaluation at increasing levels of complexity.
